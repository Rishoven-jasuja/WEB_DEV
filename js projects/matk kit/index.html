How to Use

A quick guide to every module in this toolkit

01 · Set Operations

  1. Enter elements for Set A as comma-separated values (e.g. 1, 2, 3).
  2. Enter elements for Set B similarly.
  3. Click Compute to see Union, Intersection, A−B, B−A and an animated Venn diagram.
  4. Use Clear to wipe inputs and Reset to restore defaults.

02 · Truth Table Generator

  1. Type a Boolean expression using operators: AND, OR, NOT, XOR, NAND, NOR.
  2. Variables are single uppercase letters (A–Z).
  3. Use operator quick-insert buttons below the field for convenience.
  4. Click Generate Table — all combinations are computed automatically.
  5. The output column is highlighted in gold.

03 · Permutation & Combination

  1. Enter n (total items) and r (items to choose/arrange).
  2. Click Calculate to see nPr and nCr results.
  3. Formulas and intermediate steps are displayed for clarity.
  4. A factorial reference table is also shown.
  5. Constraint: 0 ≤ r ≤ n ≤ 20.

04 · Graph Tool

  1. Add nodes by typing a name and clicking Add.
  2. Add edges by specifying From, To, and Weight (default 1).
  3. The graph renders live on the canvas. Drag nodes to reposition.
  4. Enter Source and Target nodes, then click Find Path to run Dijkstra's algorithm.
  5. The shortest path and total distance are displayed, and the path is highlighted on the canvas.
